# <font color="#C3B1E1">Système de Gestion de Catalogue et de Devis (Backend)</font>

## <font color="#C3B1E1">Description du Projet</font>
Ce projet est une application de gestion robuste développée avec **Spring Boot**, conçue dans le cadre d'un Projet de Fin d'Études (PFE). Il permet la gestion complète d'un catalogue de produits, le suivi des demandes de devis et la gestion des utilisateurs et de leurs paniers.

L'accent a été mis sur la qualité logicielle en appliquant rigoureusement les principes d'architecture moderne et les patrons de conception (Design Patterns).

## <font color="#C3B1E1">Fonctionnalités Clés</font>
- **Gestion du Catalogue** : Organisation des produits par marque et suivi de leur état.
- **Cycle de Vie des Demandes** : Gestion des demandes d'utilisateurs et génération automatique de devis.
- **Système de Panier** : Gestion persistante des sélections de produits.
- **Sécurité & Authentification** : Gestion des utilisateurs par rôles.
- **Gestion Documentaire** : Association de documents aux entités métiers.

## <font color="#C3B1E1">Architecture & Principes de Génie Logiciel</font>
Le projet met en œuvre les concepts fondamentaux de l'ingénierie logicielle :

- **Patrons de Conception (GoF)** :
    - Patrons de création
    - Patrons de structure
    - Patrons de comportement
- **Principes GRASP** (General Responsibility Assignment Software Patterns)
- **Principes SOLID**
- **Contraintes OCL** (Object Constraint Language)
- **Modèle DTO** (Data Transfer Object)

## <font color="#C3B1E1">Technologies Utilisées</font>
- **Langage** : Java
- **Framework Core** : Spring Boot
- **Accès aux données** : Spring Data JPA / Hibernate
- **Gestionnaire de dépendances** : Maven
- **Validation** : Jakarta Bean Validation & OCL manuel
